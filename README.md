# Skill Hub

OpenClaw Skill 发现 · 评测 · 筛选 · 推荐

> 只收录亲手验证过的 Skill，拒绝搬运。

## 这是什么

OpenClaw 有大量社区 Skill，但质量参差不齐，中文生态几乎空白。

Skill Hub 帮你解决四个问题：

- **发现** — 去哪找好用的 Skill
- **评测** — 这个 Skill 到底能不能用
- **筛选** — 哪个适合我的场景
- **推荐** — 哪些 Skill 搭配在一起效果好

## 评测标准

每个 Skill 从 6 个维度评分（1-5 分）：

| 维度 | 说明 |
|------|------|
| 🔧 可用性 | 能否顺利安装和运行 |
| 🎯 有效性 | 能否完成描述的功能 |
| 🛡️ 安全性 | 是否有可疑指令或风险 |
| 📖 文档质量 | SKILL.md 是否清晰规范 |
| 🔄 维护状态 | 是否持续更新 |
| 🌐 中文适配 | 是否支持中文场景 |

详细标准见 [evaluation-criteria.md](./evaluation-criteria.md)
