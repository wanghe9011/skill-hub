## 这个项目是什么

市面上有很多 OpenClaw Skill 列表，动辄 5000+ 个链接。但用户看完还是不知道该装哪个。

Skill Hub 换一种方式：

- **从你的职业出发**，告诉你需要哪些 Skill
- **给完整方案**，不只是列名字，而是怎么装、怎么配、怎么搭配使用
- **全部实测过**，每个推荐都经过真实验证
- **中文原生**，面向中文用户和中文工作场景
