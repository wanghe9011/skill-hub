## 这个项目和 awesome list 有什么不同

市面上有很多 Skill 列表，动辄 5000+ 个链接。但用户看完还是不知道该装哪个。

Skill Hub 的做法不同：

- **从任务出发**，不是列 Skill 名字，而是给你完整的解决方案
- **全部实测过**，每个推荐都经过真实验证
- **中文原生**，面向中文用户和中文工作场景
