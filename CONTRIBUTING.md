# 贡献指南

欢迎提交 Skill 评测！

## 提交评测

1. Fork 本仓库
2. 复制 `skills/_template.md`，重命名为 Skill 名称
3. 按模板填写评测内容
4. 提交 PR

## 评测要求

- **必须亲自测试过**，不接受纯搬运
- 填写完整的评分和测试记录
- 注明测试环境和日期

## 提交合集

1. 复制 `packs/_template.md`
2. 包含的每个 Skill 必须有对应的评测
3. 说明组合使用的效果

## 报告问题

- 发现 Skill 有安全隐患？提 Issue 并标注 `security`
- 评测内容有误？提 Issue 或直接 PR 修正

## 写作风格

- 中文为主
- 说人话，不写技术文档腔
- 有问题就直说，不用委婉
